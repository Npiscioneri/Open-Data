# 📊 Gender Equality Data Visualization  

Un'applicazione web realizzata con **React**, **Chart.js** e **PapaParse** per visualizzare dati sulla **parità di genere** in Europa. Il progetto permette di esplorare i dati in formato tabellare e tramite grafici interattivi.  

## 🚀 Funzionalità  
✅ **Visualizzazione dati**: Tabella dinamica con i dati grezzi.  
✅ **Grafici interattivi**: Analisi del divario di genere con **Chart.js**.  
✅ **Lettura dataset CSV**: Caricamento e parsing automatico con **PapaParse**.  
✅ **Interfaccia moderna**: Stile pulito con **Tailwind CSS**.  
✅ **Routing tra pagine**: Navigazione fluida con **React Router**.  

## 🛠️ Tecnologie utilizzate  
- **React** (con Vite) ⚛️  
- **Chart.js + react-chartjs-2** 📊  
- **PapaParse** (per leggere file CSV) 📄  
- **Tailwind CSS** 🎨  
- **React Router** 🔄
